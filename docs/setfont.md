# Command: setfont

### Change the default font

## SYNOPSYS
+ setfont fontname

## DESCRIPTION
This command overwrite the current charset with a new one

## EXAMPLES
+ setfont boldfr0
+ setfont default

## NOTES
Font files are in /usr/share/fonts
