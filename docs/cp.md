# Command: cp

### Copy a file to another file

## SYNOPSYS
+ #cp from to

## DESCRIPTION
Copy a file to another. Only 40KB can be copied. It's a software limit. cp tries to allocate 40KB, if there is not enough memory it displays Out of memory.

## EXAMPLES
+ cp from.txt to.txt
